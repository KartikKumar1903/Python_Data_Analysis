# Python_Data_Analysis
This repository documents my journey of learning **data analysis** using **Python**. It contains practice notebooks, small projects, and experiments as I learn to work with data, statistics, and visualizations.

## 📚 What I'm Learning

- **Core Python for Data Analysis:**
  - Variables, functions, loops, and data structures
  - Working with lists, dictionaries, and iterators

- **Data Libraries:**
  - **NumPy** for numerical operations and arrays
  - **pandas** for data manipulation, filtering, grouping, and merging
  - **Matplotlib** for data visualization

- **Data Analysis Skills:**
  - Loading and cleaning data (CSV, Excel, etc.)
  - Exploratory Data Analysis (EDA)
  - Descriptive statistics and summary metrics
  - Handling missing data, outliers, and transformations
  - Aggregations, groupby operations, and pivoting

- **Visualization & Insights:**
  - Creating plots (line, bar, scatter, histograms, box plots)
  - Understanding distributions and trends
  - Communicating insights from data

## 🛠 Tools & Technologies

- Python
- pandas
- NumPy
- Matplotlib
